## 발견 사항 (다음 세션 참고)

1. **Spring Boot 4.0.6 YAML 라우트 버그**: `spring.cloud.gateway.routes` YAML 설정이 로드되지 않음 → `RouteLocatorBuilder`로 프로그래밍 방식 사용
2. **YAML globalcors 미동작**: `CorsWebFilter` 빈으로 대체
3. **Gradle 8.14+ 필수**: Boot 4.0.6이 8.14 이상 요구
4. **Dockerfile CRLF**: Windows 환경에서 `dos2unix` + `bash` 필요
